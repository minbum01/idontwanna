# 합격선 통합 산출 (idontwanna)

전국 지방공무원·교육행정 **필기 합격선(커트라인)** 데이터를 한곳에 모아 검증된
표(HTML)로 보는 프로젝트. **원본 파일과 산출물을 한 저장소에** 담아, 어느 PC에서
clone 하든 바로 이어서 작업할 수 있다.

## 폴더 구조
```
(저장소 루트)
├─ 합격선 최종모음/        ← 원본 파일 243개 (읽기 전용, 절대 수정 금지)
│   └─ 01 서울 공무원 / 02 서울 교행 / ... (지역 × 공무원/교행)
└─ _통합산출/              ← 코드·데이터·결과물
    ├─ 검증_data.csv       ← ★ 사람이 확인한 최종 데이터(유일한 진실)
    ├─ build_v2.py         ← 검증_data.csv → 합격선_관리_v2.html 생성기
    ├─ 합격선_관리_v2.html  ← 결과물(브라우저로 열기)
    ├─ README.md           ← 자세한 사용법·스키마
    └─ 작업지침.md         ← ★ 작업 규칙(반드시 준수)
```

## 다른 PC에서 이어서 작업하기
```bash
git clone https://github.com/minbum01/idontwanna.git
cd idontwanna/_통합산출
pip install pdfplumber PyMuPDF Pillow openpyxl olefile
python build_v2.py          # → 합격선_관리_v2.html 갱신
```
- 원본 폴더(`합격선 최종모음`)가 저장소 안에 함께 있으므로 HTML의 "출처" 링크가 그대로 동작한다.
- 작업 방법·데이터 스키마는 **[_통합산출/README.md](_통합산출/README.md)**,
  지켜야 할 규칙은 **[_통합산출/작업지침.md](_통합산출/작업지침.md)** 참고.
